# Lead Tracker Chrome Extension

A Chrome extension for saving and managing lead URLs. Quickly save the current tab's URL or manually enter links to build a list of leads.

## Features

- Save the current browser tab URL with one click
- Manually enter and save any URL
- Persistent storage using localStorage
- Double-click to delete all saved leads
- Leads displayed as clickable links that open in a new tab

## Tech Stack

- HTML, CSS, JavaScript
- Chrome Extensions API (Manifest V3)

## How to Run

1. Open Chrome and navigate to `chrome://extensions/`
2. Enable **Developer mode** (top right)
3. Click **Load unpacked** and select this project folder
4. Click the extension icon in the toolbar to open the popup
